# Teapot Assets

## teapot.obj

A UV-mapped variant of the classic Utah teapot (Martin Newell, 1975).

- 792 vertices, 992 triangles, with normals and UVs (UV range 0..2, so the
  pattern tiles twice across the surface).
- Source: redistributed via the [phoria.js](https://github.com/kevinroast/phoria.js)
  repository (MIT licensed), originally Max-exported "Teapot01" from 3ds
  Max.
- License: the Utah Teapot model is freely available for any use including
  commercial use, per the Utah Graphics Lab terms at
  https://graphics.cs.utah.edu/teapot/. When using or redistributing, identify
  it as the **Utah Teapot** (or **Modified Utah Teapot**) and acknowledge its
  origin as the historic Utah Teapot developed at the University of Utah.

## teapot_albedo.png

A 512×512 "blue-and-white china" procedural texture: cobalt-blue floral spots
on a warm cream background.

- Source: procedurally generated by `scripts/gen_teapot_albedo.py`.
- License: CC0.

Regenerate with:

```bash
uv run python scripts/gen_teapot_albedo.py
```
